Stack(스택) 메모리
 - 함수가 실행될 때 기본적으로  확보되는 메모리
 - 크기가 상대적으로 작다
 - 함수 실행코드, 함수 파라메터, 지역변수
 - 사용하는데 속도가 빠르다.



Heap(힙) 메모리
 - 컴퓨터의 남아있는 램
 - 크기는 보통 무한으로 처리
 - 동적(프로그램 실행 중) 할당 용도로 사용 (new)
 - 속도가 느리다. (보안때문)

C#의 데이터 타입

값타입(Value Type)			참조타입 (Reference Type)
 - 스택메모리에저장			 - 힙 메모리에 저장
 - unmanaged type			 - managed type(가비지 콜랙팅이 된다.)
 - 깊은 복사(완전한 사본을 가지게 된다)	 - 얕은 복사(복사 속도가 빠르다)
 - struct				 - class
