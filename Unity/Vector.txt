Vector(벡터)
 - 힘의 방향과 크기를 나타낸다.

 벡터의 크기는 = 벡터의 길이(Lenght) = sqr(x^2+y^2+z^2)

Unit Vector(단위벡터)
 - 방향만 나타낸다.
 - 크기가 1인 벡터
 - 계산법 : 단위벡터 = 벡터 / 벡터의 크기
 - 단위벡터로 만드는 것 : Normalize (정규화) 라고한다.

 방향 벡터 = 목표위치 - 현재위치


Normal 벡터
 - 특정 평면에 수직한 벡터 -> 벡터 2개가 있다면 하나의 평면을 구할수 있고 이를 외적하면 수직한 벡터를 구할수 있다. 
									왼손으로 노말벡터위치 볼수있음
 - 반사를 계산하는데 필수적으로 사용됨

외적(Cross Product)
 - 노멀 벡터를 계산한다 => 삼각형의 앞뒤를 결정한다. 카메라가 보는 위치를 계산할수 있고 
			평면의 노멀벡터와 카메라가 보는 벡터가 180도면 정면 0도면 반대편

내적(Dot Product)
 - 백터의 사이각을 구하는데 사용할 수 있다. (1,2,3) dot (2,0,4) = 1*2 + 2*0 + 3*4 = 14
 - 내적으로 두 벡터의 각도 구하기가 됨  (각도) = acos( (a 내적 b) / (a크기 * b크기) )
